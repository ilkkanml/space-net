# PROJECT_CONTEXT.md

## Proje adı
S.P.A.C.E. NET

## Studio adı
S.P.A.C.E. NET Studio

## Proje türü
Web tabanlı 3D izometrik fabrika otomasyon, üretim zinciri, ekonomi, consortium, event, prestij ve kozmetik gelir modeli üzerine kurulu uzun vadeli endüstriyel sci-fi oyun projesi.

## Ana vizyon
S.P.A.C.E. NET, uzun yıllar geliştirilecek yaşayan bir oyun ekosistemi olarak tasarlanacaktır.

## Başlangıç teknolojisi
- HTML
- CSS
- JavaScript
- Three.js
- WebGL
- LocalStorage

## Uzun vadeli teknik hedef
- Backend API
- Veritabanı
- Oyuncu hesap sistemi
- Marketplace backend
- Canlı ekonomi servisi
- WebSocket / multiplayer altyapısı
- Admin panel
- LiveOps paneli
- Güvenlik ve log sistemi
