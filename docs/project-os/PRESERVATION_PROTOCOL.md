# S.P.A.C.E. NET Studio — Preservation Protocol

## Core Philosophy

### Must Preserve
- silence-first atmosphere
- industrial loneliness
- calm progression pacing
- observe-and-maintain gameplay feeling
- restrained E.V.A. presence
- low-intensity ambience

### Avoid
- dopamine-heavy feedback
- mobile-management pacing
- aggressive notification spam
- loud sci-fi spectacle
- companion AI feeling
- permanent HUD pressure

## UI Rules
- minimal movement
- low pulse intensity
- soft transitions
- no screen saturation

## Audio Rules
- low-frequency ambience
- restrained warning tones
- conveyor comfort priority
- silence as active design element

## E.V.A. Rules
- rare appearance
- short observational language
- no exposition-heavy dialogue
- no emotional overacting
