# WORK_SESSION_PROTOCOL.md

## Başlangıç
Kullanıcı `BOOT` yazarsa:
- CURRENT_STATE.md oku.
- ACTIVE_MILESTONE.md oku.
- DECISIONS_LOG.md oku.
- Mevcut milestone’dan devam et.

## Kapanış
Kullanıcı `HANDOFF` yazarsa:
- O oturumdaki kararları özetle.
- CURRENT_STATE.md güncelleme metni ver.
- DECISIONS_LOG.md karar kaydı ver.
- NEXT_ACTIONS.md sonraki adımları ver.
- Gerekirse CHANGELOG.md ve BUG_REPORTS.md güncellemesi hazırla.

## Karar
Kullanıcı `DECISION` yazarsa:
- Son konuyu resmi karar formatına çevir.
- Aktif milestone’a mı, backlog’a mı gireceğini belirt.
